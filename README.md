## About Fluxus

Fluxus is a Windows Roblox script executor centered on Lua injection, quick starts, and a compact desktop layout. It is meant for people who want a simple way to run scripts, explore a script hub, and keep their workflow in a single app without a bulky interface slowing things down.

The tool combines a no-key approach with auto update support, SQLite-backed local storage, and a desktop UI with multiple languages. With batch execution, a saved script queue, and an integrated debugger, Fluxus serves as a practical script hub and executor for routine Roblox scripting work.
